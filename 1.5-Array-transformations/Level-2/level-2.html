Exercise 5

Donat un array "[ 1, 3, 7, 10 ,15, 17, 11, 5, 8, 12, 9 ]", crea una funció en una sola línia que faci el següent: - Filtra els nombres majors o iguals a 10. - Multiplica cada nombre filtrat per 2. - Calcula la suma dels nombres filtrats i multiplicats per 2. - La funció ha de retornar el resultat de la suma.

const calculation = (array) =>
array
.filter((num) => num >= 10)
.map((num) => num * 2)
.reduce((accu, value) => accu + value);
