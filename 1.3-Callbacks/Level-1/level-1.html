1.3-Callbacks - Level 1

Exercici 1

Escriu una funció anomenada processar que accepti dos paràmetres: un nombre i una funció de callback. La funció processar ha d'invocar la funció de callback, passant el nombre com a paràmetre.

const processar = (num, callback) => {
return callback(num);
};

processar(12, (num) => num + num);
Exercici 2

const calculadora = (num1, num2, callback) => {
return callback(num1, num2); };

calculadora(2, 3, (num1, num2) => num1 + num2);
